Durations

Fast · 120 ms

--dur-fast · hover, tap, micro-state flips

Base · 220 ms

--dur · popovers, menus, drawer reveal

Slow · 320 ms

--dur-slow · windows, sheets, view transitions

Symbol-effect stagger

80 ms delay · bounce on each item

Rules

  • Default to --ease (standard). Use decelerate for entering elements (popovers, menus) and accelerate for dismissals.
  • Use spring only on user-initiated, direct-manipulation feedback (toggle thumbs, dock magnification).
  • Hover/press states must complete inside --dur-fast (≤ 150 ms).
  • SF Symbol effects: .bounce 600 ms / .pulse 1 s / .variableColor 1.2 s — never chain more than two on a single icon.
  • Respect prefers-reduced-motion: collapse durations to ≤ 100 ms and disable spring overshoot.
