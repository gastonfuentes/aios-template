Rules

  • Only the keyboard-focused element shows the ring — :focus-visible, not :focus. Pointer clicks never paint a ring.
  • The default ("key-equivalent") button shows --focus-ring-key from the moment the window is key; tabbing moves the ring (and key equivalence) to whatever element is focused.
  • Rings always inherit the system accent (blue by default; matches user's accent if changed).
  • Outside a window's content (toolbars, sidebars), rings are drawn at 2 px tight to avoid bleeding across hairlines.
  • Never combine focus with hover/pressed styling — focus state takes precedence on the outer shadow.
  • Respect prefers-reduced-motion: skip the 120 ms fade-in on appearance.
