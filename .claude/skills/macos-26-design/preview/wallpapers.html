Usage rules

  • Use wallpapers as the desktop / lockscreen background — never as a control fill.
  • Don't tile, crop tightly, or apply additional gradients; the compositions are tuned end-to-end.
  • When pulling colors for chrome, sample from the wallpaper's opposite tonal end so Liquid Glass picks up contrast.
  • For light mode prefer Sequoia / Ventura; for dark prefer Aurora / Monterey.
