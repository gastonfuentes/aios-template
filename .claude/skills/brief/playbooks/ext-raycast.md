# Playbook: ext-raycast

## Targets obligatorios
- **Raycast API surface**: List, Form, Detail, ActionPanel, preferences, LocalStorage.
- **Command modes**: `view` (UI) vs `no-view` (instant actions) vs `menu-bar`.
- **Distribucion**: Raycast Store (free), submission review.
- **AI commands**: nueva categoria 2024+, requiere Raycast Pro ($8/mes usuarios finales).
- **Bundle size**: Raycast es estricto con size — evitar libs gigantes.

## Targets opcionales
- **Preferences schema**: types soportados, pregunta al user al install.
- **Node-only**: algunas libs npm funcionan (fs, child_process).
