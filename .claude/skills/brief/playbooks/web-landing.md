# Playbook: web-landing

## Targets obligatorios
- **Competidores directos**: identificar 3–5 landings del mismo nicho, capturar estructura (secciones, CTAs, pricing).
- **Patrones de conversion**: hero arriba del fold, social proof alto, CTA repetido, faq al final.
- **Core Web Vitals**: LCP < 2.5s, CLS < 0.1, INP < 200ms como objetivo.
- **SEO tecnico**: meta tags, OG image, schema.org Organization + WebSite.

## Targets opcionales
- **A/B testing tool**: Vercel Edge Config, PostHog, GrowthBook.
- **Heatmaps / session replay**: PostHog, Microsoft Clarity (gratis).
